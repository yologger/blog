# Mongo DB 정리
비필수 스키마와 함께 JSON과 같은 문서를 사용하여 대량의 데이터를 저장하는 NoSQL 데이터베이스입니다. 

## Mongo DB 설치
`Mac OS`에서는 `Homebre`로 Mongo DB를 설치할 수 있다.
``` shellsession
$ brew tap mongodb/brew

$ brew install mongodb-communit
```

## Compass 설치
`Compass`는 Mongo DB를 위한 Database Client다.
